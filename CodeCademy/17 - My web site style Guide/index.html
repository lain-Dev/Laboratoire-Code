font-family

permet de définir une liste, ordonnée par priorité, de polices(serif, cursive, Georgia...)

font-style

permet de sélectionner une fonte italique (italic) ou (oblique)

font-size

définit la taille de fonte utilisée pour le texte.(12px, 80%, smaller...)

font-weight

permet de définir la graisse utilisée pour le texte(bold, bolder, 100)

font-variant

propriété raccourcie représentant les propriétés : font-variant-caps, font-variant-numeric, font-variant-alternates, font-variant-ligatures et font-variant-east-asian
